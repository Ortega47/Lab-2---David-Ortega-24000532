Ejercicio 1: RPG Character Sheet

En este ejercicio se desarrolló una pantalla de creación de personaje estilo RPG utilizando Jetpack Compose.
La aplicación muestra tres estadísticas del personaje: Vitality, Dexterity y Wisdom, cada una con un botón que permite generar un valor aleatorio.

Los valores de las estadísticas se suman automáticamente para mostrar un puntaje total.
Dependiendo del total, la app muestra un mensaje de retroalimentación:

Si el total es bajo, recomienda volver a lanzar los dados.

Si el total es alto, muestra un mensaje indicando que el personaje es muy poderoso.

El ejercicio se enfocó en el uso de Composables reutilizables, manejo de estado y organización visual con filas y tarjetas.



Ejercicio 2: Traffic Light Simulator

En este ejercicio se creó una simulación de un semáforo utilizando Jetpack Compose.
La pantalla muestra tres luces (roja, amarilla y verde) alineadas verticalmente.

El semáforo cambia de color automáticamente, sin interacción del usuario, siguiendo un ciclo realista:

Rojo durante 2 segundos

Verde durante 2 segundos

Amarillo durante 1 segundo

La lógica se implementó usando estado y LaunchedEffect, lo que permite que el ciclo se ejecute de forma continua y segura, respetando el ciclo de vida de la aplicación.

Este ejercicio sirvió para practicar manejo de estado, efectos en Compose y actualización automática de la interfaz.
